Sketchy

Made for CSCI 0150: Introduction to Object-Oriented Programming and Computer Science (Fall 2019)

This was my final project for my first CS class at Brown and my longest yet; 23 classes and 1,865 lines of code. Our assignment was to build a graphics editor solely based on Java and JavaFX. Below are some features from the project. Not shown (but also functional) is functionality to save/load creations for future work.


Translate

Using the "Select Shape" option, user can move shapes around the canvas. Translation is controlled by dragging the mouse across the screen.

Raise/Lower

The "Raise" and "Lower" buttons can move shapes up or down in the list of layers. Layers are preserved even after saving/loading.

Draw with Pen

Users can select a color and draw on the canvas with a pen controlled by mouse input. Works with undo and redo stacks.

Undo/Redo

Using two stacks, Sketchy can undo and redo all of its commands, including creating, deleting, rotating, translating, filling, and resizing shapes.

Resize

By using Shift-Click, users can select objects and resize them to be larger or smaller. Users can undo/redo resizing as well.

Fill

By selecting shapes and a color with built-in color picker, Sketchy can change the fill of any of its shapes.

Rotate

By using Ctrl-Click, users can use Sketchy to rotate ellipses and rectangles by any amount, and can undo/redo these changes.

Delete

By using "Select Shape," Sketchy can delete any rectangle or ellipse previously added to the canvas.
