Graph

Made for CSCI 0160: Introduction to Algorithms and Data Structures (Spring 2020)

In this project we modeled the graph data structure with an adjacency matrix (a 2D array), added most basic functionality, and built a modified version of Google's PageRank algorithm. The program functions for both directed and undirected graphs. Below are demonstrations of some of the features I implemented:


PageRank

In a directed graph, the PageRank algorithm ranks pages by their importance. This is calculated by using the "Rank" of all of the pages that point to it.

Incoming Edges

By clicking on any vertex, Graph can use the adjacency matrix it is built on to return a list of all of the edges pointing to said vertex.

Are Adjacent

By selecting any two vertices, Graph will search through its underlying adjacency matrix and return true if an edge connects them.

Connecting Edge

After selecting two vertices, instead of just determining whether or not an edge exists, Graph can return the weight of that edge.

End Vertices

By clicking on an edge, Graph can return the two vertices that are associated with said edge.

Opposite

Upon selecting an edge and an associated vertex, Graph will return the vertex opposite that edge.

Vertices/Edges

Graph can use its underlying adjacency matrix to return lists of all of its vertices and all of its edges (which are double for undirected graphs).
