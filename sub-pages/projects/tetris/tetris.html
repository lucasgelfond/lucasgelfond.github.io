Tetris

Made for CSCI 0150: Introduction to Object-Oriented Programming and Computer Science (Fall 2019)

For this project we used Java and JavaFX, more specifically a 2D array and a Java timeline to replicate the classic game of Tetris. Functionality includes line-clearing, randomized pieces, play/pause, and a quit button.

This ended up being one of the hardest projects of the course and took way longer than expected to debug. Some of the delegation and general structure/design of the program felt extremely unfamiliar.

Regardless, this project was a blast to build, even more fun to test, and I think was a great way of paying homage to a game so integral to computer graphics.
